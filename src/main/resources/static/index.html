About Us

UPTD SDN 3 Jatibarang is a state elementary school institution located at Jalan Mayor Dasuki, Kab. Indramayu.

This public elementary school was founded in 2016. Currently UPTD SD Negeri 3 Jatibarang uses the 2013 elementary school learning curriculum. UPTD SDN 3 Jatibarang has a principal named Sukirno assisted by an operator named Angga Novanto L.

UPTD SDN 3 Jatibarang accreditation
UPTD SDN 3 Jatibarang has grade A accreditation with a score of 92 (2018 accreditation) from BAN-S/M (National Accreditation Body) for Schools/Madrasahs.
